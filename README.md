# 🌱 AI Crop Disease Detection System

This project is an end-to-end AI-powered solution designed to detect crop diseases from leaf images and provide instant treatment recommendations.

Built using Vision Transformer (ViT), PyTorch, and Hugging Face, the system delivers accurate predictions through an easy-to-use Gradio web interface.

It aims to support farmers by enabling early disease detection, reducing crop loss, and promoting sustainable agriculture.
